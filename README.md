###qNode v 0.1.0

This script quickly initializes a webserver with the following frameworks:
- git
- node.js
- express.js
- jasmine.js
- ejs
- psql
- sequelize
- nodemon


##How to use:

Simply run `qNode.sh` in the parent directory of the future project, and follow
the onscreen instructions.`
